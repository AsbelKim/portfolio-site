# Asbel Kimutai — Portfolio Website

Personal portfolio for **Asbel Kimutai**, Software Engineer, ERP Specialist & AI Builder based in Nairobi, Kenya.

**Live site:** [AsbelKim.github.io/portfolio-site](https://AsbelKim.github.io/portfolio-site)

## Sections

- **Hero** — title, summary, social links, CV download
- **About** — background, stats
- **Skills** — Languages & Frameworks, Backend & Integrations, AI & Prompt Engineering, ERP Systems, Deployment & Tools
- **Experience** — Imara Logic Systems, Safaricom PLC, Esra Company, ERP Business System Solutions
- **Projects** — Sokoni Connect, AI-Powered Business Automation
- **Education & Certifications** — BSc Software Engineering, TallyPrime Specialist, Prompt Engineering & AI Skills, MB-920 (In Progress)
- **References** — 3 professional references

## Stack

- Plain HTML, CSS, JavaScript — no build step required
- Scroll-reveal animations via `IntersectionObserver`
- Particle canvas background (vanilla JS)
- Fully responsive (mobile-first)

## Structure

```
portfolio-site/
├── index.html        — main portfolio (all sections)
├── styles.css        — design system & layout
├── script.js         — navbar, scroll effects, animations
├── assets/
│   └── cv.html       — printable CV (opens in new tab)
└── images/
    ├── hero.jpg      — professional headshot (hero circle)
    └── about.jpg     — portrait for About section
```

## Run locally

Open `index.html` directly in your browser, or use the VS Code **Live Server** extension for hot-reload preview.

## Deploy to GitHub Pages

1. Push this repo to GitHub as `portfolio-site`
2. Go to **Settings → Pages → Source → main branch / root**
3. Your site will be live at `https://AsbelKim.github.io/portfolio-site`
